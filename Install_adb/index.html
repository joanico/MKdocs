Install Apps with adb on Ubuntu¶

Step to add apps or send apps from local server to Laptop and install app to tablet

Access to local server¶

  1. Connect to server local or local LAN like : CatalapLAN

  2. Access to the site local like : http://crapple.cat:8080/

  3. Download apps : apps_release.apk

Manual configuration¶

  1. Put apps on Download or Desktop directory.
  2. Connect your andoid device in to laptop.
  3. Open the terminal on ubuntu.
  4. Using adb to enable USB debugging on your Android Device.

If not have adb on OS ubuntu you need to install it.

  1. Install adb in to OS ubuntu sudo apt install android-tools-adb android-tools-fastboot
  2. After install adb you can run adb devices in terminal add devices to see your list of devices.
  3. Install adb install -r -g ~/Downloads/app-release.apk ➜ ~ adb install -r -g ~/Downloads/app-release.apk 3803 KB/s (1515143 bytes in 0.389s) pkg: /data/local/tmp/app-release.apk Success ➜ ~ Note : If you install get an error like :

~ adb install -r -g ~/Downloads/app-release.apk 3744 KB/s (1515143 bytes in 0.395s) pkg: /data/local/tmp/app-release.apk Failure [INSTALL_FAILED_UPDATE_INCOMPATIBLE]
