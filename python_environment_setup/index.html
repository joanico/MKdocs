Python - Hahu no Kria Enviroment¶

Python disponivel iha variasaun bo’ot hosi plataforma inklui Linux no Mac OS X. Iha ne’e oinsá atu setup ita-nia Python enviromentu.

Lokál Setup Enviromentu

Loke terminál iha Linux ka Mac Os no hakerek python atu hatene rezultadu ita instala tiha ona no versaun ne’ebé instala ona.

Asesu situs Python¶

Python atualiza ona source code no dokumentu foun sira ne’ebé disponivel ona iha ofisiál website hosi Python https://www.python.org

Ite bele download dokumentu python iha python doc. Dokumentu sira ne’e disponivel iha HTML, PDF no PostScript formats.

Instalasaun Python¶

Distribuisaun Python disponivel iha variedade boot hosi plataforma sira. Ita presiza ba download kódigu bináriu balu aplikável ba iha ita-nia plataforma no instalasaun Python.

Unix no Instalasaun Linux

Iha Linux python eziste ona maibé ne’e sei ho versaun tuan atu atualiza versaun foun iha maneira oin-oin iha ne’e hatudu de’it maneira ida mak ho etapa simples atu halo instalasaun Python iha Unix/Linux machine.

  • Loke Web browser no ba iha download python

  • Halo tuir link download zipped source code disponivel ba Unix/Linux.

  • Download no extract filé. Edita módulu/setup file ne’ebé mak ita hakarak ba kostume opsaun balu.

  • Run ./configure script

  • Make

  • Make install

Instalasaun Python ne’e iha lokasaun standar /usr/local/bin no libraria sira iha /usr/local/lib/pythonXX iha ne’ebé XX iha versaun Python.

Instalasaun Windows Iha ne’e mak etapa atu instala Python ba iha Windows machine.

  • Loke Web browser no ba iha download python
  • Halo tuir link ba iha instalasaun Windows python-XYZ.msi dokumentus iha ne’ebé XYZ iha versaun ita presiza halo instalasaun.
  • Atu usa instalasaun iha ne’e python-XYZ.msi, sistema Windows suporta Microsoft Installer 2.0. Rai dokumentu instalasaun ba ita-nia lokál machine no run ida ne’e atu hetan nia rezultadu se nia machine suporta MSI.
  • Run dokumentu download. Ne’e lori instalasaun Python wizard, ne’ebé realiza fasil atu uza. Foin aseita iha defaul settings, hein to instalasaun hotu, no ita remata.

Instalasaun Macintosh

Macintosh eziste ona Python. Maibé ne’e dala ruma sei versaun tuan ita presiza upgrade ba versaun foun. Hare download python iha instrusaun ne’e hetan versaun agora ho extra tools ba suporta dezenvolve iha Mac. Iha folder Mac OS’s molok Mas OS X 10.3(realiza iha 2003)

Jack Jansen mantein ida ne’e no ita bele asesu hotu atu hatama dokumentu ba iha nia website - download python mac Ita bele hetan detallu instalasaun kompletu ba instalasaun Mac OS.
